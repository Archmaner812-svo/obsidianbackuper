## English Description

Obsidian Backuper is a blazing fast and reliable utility written in Rust designed to protect your "Second Brain". It automates the process of backing up your Obsidian vaults into compressed archives, ensuring data integrity with the speed and safety only Rust can provide.

### ‚ú® Key Features
* Blazing Fast: Written in pure Rust for maximum performance and minimal resource usage.
* Standalone Binary: No need to install Python, Node.js, or any other runtime. Just a single executable file.
* Smart Compression: Efficiently archives your notes and attachments to save disk space.
* Safe & Secure: Leverages Rust's memory safety guarantees to prevent data corruption during backup.

3.  Run the application.
4.  Follow the prompts to select your vault and backup location.
